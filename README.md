# The Flatiron School Design System

## Installation & Usage

### Install

`yarn add @flatiron-school/ui`

### Usage

```jsx
  import { Button } from '@flatiron-school/ui'

  export default () => {
    return <Button primary>Foo</Button>
  }
```
